# 平台配置

## 目录

平台配置
├── 应用管理（含菜单树、权限树维护）
├── 菜单管理（后台，已迁移 — 后台菜单入口应隐藏）
├── 权限管理（后台，已迁移 — 后台菜单入口应隐藏）
├── 角色管理（后台）
│ ├── 基本信息
│ ├── 权限绑定（菜单/API）
│ ├── 默认数据权限类型（可选）
├── 字典管理（Dictionary）
├── 参数配置（Config）
├── 国际化（i18n，可选）

## 应用中心化说明

应用管理已切换为应用中心化模型：在「应用管理」列表中以 Grid 卡片展示应用，进入详情后可维护基本信息、应用内菜单树与应用内权限树。

原独立的「菜单管理」「权限管理」页面代码仍保留供排查，但**不应再作为日常维护入口**；请通过后台菜单配置隐藏对应路由。
